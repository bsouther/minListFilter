jquery.minListFilter

A minimal jquery filter plugin for HTML select lists.

  • Download
  • GitHub
  • Ben Souther

DOM Effects

For each list in the wrapped set

  • Adds the size attribute to the select list
  • Adds an input field before the list
  • Wrapps the list and the input field in a div

Behavior

  • As you type in the added input field, options that don't match are removed from the list"
  • Once the list has been filtered down to a single option, that option is selected.

Usage

In your HTML, create a select list

        <select id="mySelectList">  
        <option value="1">One </option>
        <option value="2">Two </option>
        <option value="3">Three </option>
        </select>

In your ready function... 

        $(function(){
          $('#mySelectList').minListFilter();
        });

Options

  • size
    Sets the size attribute of the select list. Defaults to: 5
  • divWrapperClass
    The class name of the generated div that wraps the select list and generated input field. Defaults to: "selectFilterWrapper"
  • criteriaFieldClass
    The class name of the generated input field. Defaults to: "criteriaFieldClass"
  • onNarrowedToOne
    A function that will be called once the list has been narrowed down to one option Defaults to nothing" 
      $(function(){
        $('#mySelectList').minListFilter(
          {size:               10,
           divWrapperClass:    "mySelectListWrapper",
           criteriaFieldClass: "myCriteria",
           onNarrowedToOne:    function(){
                                 // Move to the next field once we've narrowed
                                 // the list to a single option.
                                 $('#nextField').focus();
                               }});
      });
